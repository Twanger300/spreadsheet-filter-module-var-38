# Руководство пользователя

1. Подготовьте таблицу CSV или XLSX с заголовками в первой строке.
2. Запустите команду `sheet-filter`, указав входной и выходной файл.
3. Добавьте один или несколько параметров `--filter COLUMN OPERATION VALUE`.
4. При необходимости задайте сортировку через `--sort COLUMN` и `--desc`.
5. Проверьте сформированный файл результата.

Примеры операций:

- `equals` — точное совпадение;
- `contains` — поиск подстроки;
- `gte` — больше или равно;
- `lte` — меньше или равно;
- `between` — диапазон значений через `..`, например `10000..50000`;
- `not_empty` — непустое значение.
